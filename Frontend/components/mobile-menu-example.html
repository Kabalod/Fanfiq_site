1. Обзор проекта

Название проекта

Fanfiq

Универсальный поиск и читалка фанфиков

Ключевая концепция

"Spotify для фанфиков"

Единая точка входа для миллионов читателей

Fanfiq — это инновационная поисковая система и универсальная читалка фанфиков, которая объединяет контент с самых популярных платформ (Ficbook, AO3, Wattpad и др.) и делает его удобным для поиска и чтения в одном приложении.

2. Проблема и решение

⚠️ Текущая проблема

  • • Фанфики разбросаны по десяткам разных сайтов
  • • У каждой платформы неудобный и ограниченный поиск
  • • Читателям приходится вручную проверять несколько платформ
  • • Отсутствует единый интерфейс для чтения

✅ Наше решение

  • • Агрегация фанфиков в единую базу данных
  • • Быстрый и мощный полнотекстовый поиск
  • • Лучший в классе ридер для комфортного чтения
  • • Единое приложение для всех потребностей читателей

4. Техническая архитектура

Поток данных

Frontend (Next.js)
→
API (FastAPI)
→
Workers (Celery)
→
DB (PostgreSQL + Redis)

Ключевые особенности

  • • Модульная архитектура (monorepo) — легкое масштабирование системы
  • • Быстрое добавление парсеров — новые сайты без остановки сервиса
  • • Высокая производительность — готовность к большим нагрузкам
  • • Автоматический деплой — Docker + Railway
